# UncleCode Provenance

This directory records the release-surface provenance for the UncleCode rebuild.

- `manifest.json` is the machine-checkable source of truth used by release-surface tests.
- Status values are limited to `rewritten`, `clean-room-adapted`, `licensed-reuse`, and `inspiration-only`.
- Every major subsystem listed in `.sisyphus/plans/unclecode-platform-rebuild.md` should map to one manifest entry before release.
